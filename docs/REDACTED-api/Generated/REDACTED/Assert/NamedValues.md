# Assert.NamedValues&lt;T,T1&gt; method

Creates an action that asserts that the named value, stored in the test case context, is as expected (compared to another named value, also stored in the test case context).

```csharp
public static AssertContextValues<T, T1, HearingInstrumentContext> NamedValues<T, T1>(
    string firstItem, AssertType relation, string secondItem, int firstVectorIndex = -1, 
    int secondVectorIndex = -1, Func<T1, T> manipulateLimitFunc = null)
    where T : IComparable<T>
```

| parameter | description |
| --- | --- |
| firstItem | First named value, stored in the test case context |
| relation | Type of relation to assert |
| secondItem | Name of the expected (equals) / undesired (not equals) / limit value to compare with (less than/greater than a.s.o), which also is stored in the test case context |
| firstVectorIndex | The vector index of the first item in the test case context. (Default: -1, means: not used) |
| secondVectorIndex | The vector index of the second item in the test case context. (Default: SameAsActualVectorIndex). If the actual value is a vector, but the value to compare with is NOT a vector, use the value NoLimitVectorIndex |
| manipulateLimitFunc | The function to use to manipulate the limit value stored in the test case context before the compare |
